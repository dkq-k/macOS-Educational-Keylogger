# macOS Educational Keylogger

**보안 교육용 키로거 개발 프로젝트**  
(Phase 1~3 완료 + 팀 특별 기능 포함)

이 프로젝트는 악성 키로거의 동작 원리를 이해하고, 이를 통해 효과적인 **방어 전략**을 수립하기 위한 **교육용 데모**입니다.  
**타인 기기에 무단으로 설치·사용하는 것은 법적으로 금지**되어 있습니다.

## 프로젝트 개요

- **개발 환경**: macOS + Python 3 + pynput
- **개발 목적**: 키로거 내부 구조 분석 → 공격 기법 이해 → 방어 전략 연구
- **개발 단계**:
  - **Phase 1**: 로컬 키 감지 엔진 구축 (실시간 콘솔 출력)
  - **Phase 2**: 로그 파일 시스템 구현 (타임스탬프 + 특수키 가독성 처리)
  - **Phase 3**: 은닉 및 자동화 (콘솔 숨김, 백그라운드 실행)

##  팀만의 특별한 기능

**활성 창 제목 로깅 기능** ← **팀 차별점**
- 키 입력 시 현재 포커스된 프로그램 창 이름(Google Chrome, Terminal, Safari 등)을 함께 기록
- 단순 키 기록이 아닌 **입력 맥락(Context)**까지 분석 가능하여 분석 정확도 대폭 향상
